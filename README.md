# food-client

## Запуск проекта локально

1. **Клонируйте репозиторий:**

   ```
   git clone https://github.com/aidagalimova/food-client.git
   cd food-recipes
   ```

2. **Установите зависимости:**

   ```
   yarn
   ```

3. **Переименуйте .env.example в .env**

4. **Запустите проект в режиме разработки:**

   ```
   yarn dev
   ```
